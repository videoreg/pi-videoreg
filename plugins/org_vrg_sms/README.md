# org_vrg_sms

Плагин получения и отправки SMS через модем (ModemManager D-Bus).

## Хранение SMS

Полученные SMS сохраняются как JSON-файлы в директории `~/.videoreg/sms/` (или в пути из `path.sms` манифеста).

**Имя файла:** `YYYY-MM-DD_HH-MM-SS_<номер>.json`
Пример: `2026-03-24_14-30-45_79991234567.json` (символ `+` из номера убирается)

**Структура файла:**
```json
{
  "number": "+79991234567",
  "text": "Текст сообщения",
  "timestamp": "2026-03-24T14:30:45.123456"
}
```

## SMS-команды

Если входящий SMS начинается с `/`, плагин выполняет соответствующую API-команду (только от разрешённых номеров). Ответ отправляется обратно по SMS.

Разрешённые номера берутся из `plugin_fields.org_vrg_sms.phone` файла `users.json`.
